코뮤

코뮤니티는 코딩의 신 코뮤가 만든 코딩 공부 커뮤니티이다. 코딩 독학 콘텐츠가 있으며, 모여서 각자 코딩 하는 모각코가 매우 인기가 많다.

코뮤니티 모각코

모각코라는 말을 들어보았는가? 모각코는 모여서 각자 코딩의 줄임말이다. 모각코의 커리큘럼을 설명해 보겠다.

🐍 파이썬 기초 문법

(TMI) 파이썬 이름의 의미가 뱀이기 때문에 뱀 아이콘을 붙였다.
  • 코딩의 ㅋ, 파이썬의 ㅍ조차 모르는 분들을 위한 과정
  • 파이썬 개발 환경 설치, 연산자, 자료형, 제어문, 함수까지의 커리큘럼
  • 기초 문법 개념과 문제 풀이 위주의 커리큘럼

🦞 파이썬 크롤링

(TMI) crawl은 게니까 게 아이콘이다.
  • 파이썬 기초 문법을 아는 분들을 위한 활용 과정
  • 웹 사이트에서 데이터 가져오기 (크롤링)
  • 프로젝트 위주의 커리큘럼
  • 크롤링이란 = 웹 상에 있는 데이터를 탐색하고 원하는 대로 동작시킬 수 있는 프로그램

🥐 C 언어 기초 문법

(TMI) C와 닮은 아이콘을 찾았다. 이얏호!
  • 코딩의 ㅋ, C 언어의 C조차 모르는 분들을 위한 과정
  • C 언어 개발 환경 설치, 연산자, 자료형, 제어문, 함수까지의 커리큘럼
  • 기초 문법 개념과 문제 풀이 위주의 커리큘럼

🕹 C 언어 심화 문법 + 게임 만들기

(TMI) 게임은 역시 게임 아이콘!
  • C 언어 기초 문법을 아는 분들을 위한 심화 과정
  • 구조체, 배열, 포인터 등 심화 문법을 배우고 이를 활용하는 게임을 만들어 보는 커리큘럼

🕸 웹 개발 기초 (html, css)

(TMI) WWW의 약자는 World Wide Web!
  • 웹 개발을 배우고 싶은 분들을 위한 아주아주 기초 과정
  • html, css의 기초 지식과 개념을 이해하고 간단한 구현을 해보는 과정
